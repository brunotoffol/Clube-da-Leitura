# Clube da Leitura

![](https://imgur.com/QcFYo5f.gif)

## Introdução

Um sistema de gerenciamento para um Clube da Leitura, feito como trabalho avaliativo para a Academia do Programador 2025.

### Funcionalidades
* Cadastro, Edição, Exclusão e Visualização dos amigos participantes do Clube da Leitura.
* Cadastro, Edição, Exclusão e Visualização das caixas organizadoras de revistas.
* Cadastro, Edição, Exclusão e Visualização das revistas pertencentes ao Clube da Leitura.
### Como utilizar:
1. Clone o repositório ou baixe o código fonte
2. Abre o terminal ou prompt de comando e navegue até a pasta raiz
3. Utilize o comando abaixo para restaurar as depências do projeto

```
dotnet restore
```
4. Em seguida, compile a solução utilizando o comando:
```
dotnet build --configuration Release
```
5. Para Executar o projeto compilando em tempo real
```
dotnet run --project ClubeDaLeitura.ConsoleApp
```
6. Para executar o arquivo compilado, navegue até a pasta ./ClubeDaLeitura.ConsoleApp/bin/Release/net8.0/ e execute o arquivo:
```
ClubeDaLeituraConsoleApp.exe
```

## Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.
